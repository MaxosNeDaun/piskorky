# Příspěvky k projektu

Děkujeme za zájem o přispění k projektu Gomoku Online!

## Jak přispět

### Hlášení chyb

1. Ověř, že chyba ještě nebyla nahlášena
2. Otevři nový issue s popisem:
   - Co se stalo
   - Co jsi očekával/a
   - Kroky k reprodukci
   - Prohlížeč a verze

### Návrhy funkcí

1. Otevři issue s labelem `enhancement`
2. Popiš novou funkci a proč by byla užitečná
3. Počkej na zpětnou vazbu od maintainerů

### Pull Requesty

1. Forkni repozitář
2. Vytvoř novou větev: `git checkout -b feature/moje-funkce`
3. Proveď změny
4. Commitni: `git commit -m 'Přidána nová funkce'`
5. Pushni: `git push origin feature/moje-funkce`
6. Otevři Pull Request

## Standardy kódu

- HTML: 4 mezery odsazení
- CSS: BEM metodologie pro třídy
- JS: camelCase, komentáře v češtině
- Responzivní design: mobile-first

## Testování

Před odesláním PR:
- [ ] Otestuj na mobilním zařízení
- [ ] Otestuj různé prohlížeče (Chrome, Firefox, Safari)
- [ ] Ověř, že Realtime synchronizace funguje

## Kontakt

Pro dotazy otevři issue nebo kontaktuj maintinery.
